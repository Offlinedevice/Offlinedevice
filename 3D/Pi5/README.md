## Case and docking station. Case is for:
 - Raspberry Pi 5
 - RTC battery
 - PN532 RFID module V3 
 - 5V 3007 Fan PWM Speed Adjustable CPU Cooling fan
 - Pi camera module 1, 2 or 3

## Wiring schematics Raspberry Pi 5
Make sure to use the correct Lid as the camera modules differ in height. Use 2 mm screws with the right lenght.
