## Case and docking station for:
 - Raspberry Pi Zero 2W
 - RTC DS1307
 - RC522 RFID module
 - Pi camera module 1, 2 or 3

## Wiring schematics Raspberry Pi Zero 2 W:
Make sure to use the correct Lid as the camera modules differ in height. Use 2 mm screws with the right lenght.
